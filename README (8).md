## Overview

Silicon tracking detectors in high-luminosity particle accelerators such as the
HL-LHC at CERN accumulate radiation damage over their operational lifetime.
Radiation-induced bulk defects increase leakage current, degrade charge
collection efficiency, and limit detector longevity. Accurate estimation of the
accumulated fluence from electrical measurements is therefore critical for
detector monitoring and predictive maintenance planning.

This project develops a physics-informed Machine Learning pipeline that
predicts neutron-equivalent fluence (neq/cm²) from Current-Voltage (I-V)
characteristics of silicon sensors. The model is trained on synthetic data
generated from the Hamburg radiation damage model and validated against three
independent published experimental datasets spanning 1998–2018.

## Methodology

### 1. Baseline I-V data

An un-irradiated silicon sensor I-V curve was used as the physical baseline:

> Bhatt, H. et al. (2024). *Elimination of dead layer in silicon particle
> detectors via induced electric field based charge collection.*
> Zenodo. https://doi.org/10.5281/zenodo.13683410 (CC BY 4.0)

The dataset provides leakage current density [nA/cm²] vs. reverse bias
voltage (0–100 V).

### 2. Physics-informed data augmentation

Radiation damage is modelled via the Hamburg model:

```
Delta_I(20C) = alpha * Phi * V_sensor

alpha = 4.0e-17 A/cm   (NIEL damage rate constant at 20 deg C)
Phi   = neutron-equivalent fluence [neq/cm^2]
```

Temperature dependence follows the Arrhenius relation for irradiated silicon:

```
I(T_ref) = I(T_meas) * (T_ref/T_meas)^2 * exp[-E_eff/(2*k_B) * (1/T_ref - 1/T_meas)]

E_eff = 1.21 eV   (Chilingarov 2013 — effective bandgap for irradiated silicon)
T_ref = 293.15 K  (20 deg C reference temperature)
```

5000 synthetic sensors were generated by randomising all relevant conditions:

| Parameter | Range |
|---|---|
| Fluence | 0 to 1×10¹⁵ neq/cm² |
| Temperature | -20 to +25 °C |
| Sensor volume | 0.01 to 0.2 cm³ |

Randomising temperature and volume prevents the model from memorising a fixed
measurement condition and forces it to learn physically invariant features.

### 3. Physics normalisation

Before feature extraction, all currents are normalised through two steps:

1. **Temperature correction** (Arrhenius, E_eff = 1.21 eV): scales each
   measurement to its equivalent at T_ref = 20 °C. This is applied
   consistently across all training and validation scripts.

2. **Volume normalisation**: divides by sensor volume to obtain volumetric
   current density [A/cm³], making features comparable across sensors of
   different geometries.

### 4. Feature extraction

Three physically motivated features are extracted per sensor:

| Feature | Description | Units |
|---|---|---|
| `norm_density_max_V` | Normalised current density at 100 V | A/cm³ |
| `norm_density_mid_V` | Normalised current density at 50 V | A/cm³ |
| `norm_slope` | (density_max − density_mid) / (100 − 50) | A/(cm³·V) |

### 5. Machine learning model

A `RandomForestRegressor` (100 estimators, max depth 8) was trained on 80%
of the 5000 synthetic devices and evaluated on the remaining 20%.

## Results

### Synthetic test set

| Metric | Value |
|---|---|
| R² score | 0.9989 |
| RMSE | ~1.5×10¹³ neq/cm² |

**Note on R²:** The high score reflects the deterministic `ΔI ∝ Φ` relation
embedded in the augmentation, which the model successfully inverts. This
validates the normalisation pipeline and end-to-end methodology, but does not
indicate performance on real data.

### Real-data validation

To assess generalisation, the trained model was tested against three
independent published experimental datasets. I-V data were extracted from
published figures using WebPlotDigitizer v4. Measurement conditions were
taken from the corresponding publications.

| Dataset | Technology | T (°C) | Expected fluence | Relative error | Verdict |
|---|---|---|---|---|---|
| Chen, UCSC (2016) | 50 µm ultra-thin UFSD | -20 | 1.00×10¹⁵ neq/cm² | **3.3 %** | Success |
| Scharf, Univ. Hamburg (2018) | 285 µm n-in-p strip | -30 | 8.87×10¹⁴ neq/cm² | **9.7 %** | Success |
| Riedler, Univ. Wien (1998) | Legacy strip (late-90s fab.) | -10 | 1.80×10¹⁴ neq/cm² | **137.7 %** | Failure — see below |

### Failure analysis: Riedler (1998)

The significant error on the 1998 dataset is attributed to **hardware domain
shift** arising from technology evolution. Silicon strip sensors fabricated in
the late 1990s had substantially higher surface leakage currents due to
processing limitations of the time (surface damage, interface traps, guard
ring design). The model, trained on modern sensor physics, cannot distinguish
this technology-specific noise from bulk radiation damage and consequently
overestimates the fluence.

This result is scientifically informative: it demonstrates the boundaries of
the model's applicability and confirms that temperature-normalised volumetric
density alone is insufficient to generalise across fabrication generations. A
future improvement would incorporate a technology-era feature or retrain on
legacy sensor data.

## How to run

```bash
# Install dependencies
pip install -r requirements.txt

# 1. Generate the augmented synthetic dataset (5000 sensors)
python generate_data.py

# 2. Train the model and evaluate on the synthetic test set
python train_model.py

# 3. Validate against real experimental data (3 independent datasets)
python test.py               # Riedler (1998)
python test_2016_data.py     # Chen, UCSC (2016)
python test_2018_data.py     # Scharf, Univ. Hamburg (2018)

# 4. Run the simulated real-time monitoring system
python live_monitor.py
```

> **Note:** data files must be placed in a `data/` subdirectory relative to
> the script location before running.

## Physical background

**Radiation damage in silicon:** High-energy particles displace lattice atoms,
creating stable defect clusters that act as generation-recombination centres.
The resulting increase in bulk leakage current is proportional to fluence and
sensor volume, following the Hamburg model (Moll 1999).

**Temperature dependence:** Leakage current scales exponentially with
temperature via the Arrhenius relation. Detectors at the LHC operate at
approximately −20 °C to suppress dark current and extend sensor lifetime.

**HL-LHC context:** Inner tracker layers of CMS and ATLAS are expected to
accumulate fluences of 10¹⁵–10¹⁶ neq/cm² over the full HL-LHC run. Accurate
fluence monitoring is essential for planning sensor replacement and maintaining
data quality throughout the detector lifetime.
